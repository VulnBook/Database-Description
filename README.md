# Database Description of VulnBook

This is an open-source project and this application is for researching web vulnerabilities and discussing ways to mitigate these vulnerabilities.

## Tech Stack
- Python
- Flask
- Postgresql
- MongoDB
- Jinja 2 Template
- HTML, CSS, Bootstrap

## Database
Tables:

- User
- Post
- PostImage
- Like
- Comment (Reply)
- Tag
- Mention
- BookMark
- Follow
- Notification
- OTP
- Report

### Planned Tables
- Views
